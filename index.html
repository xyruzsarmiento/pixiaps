⚡ Connect to Pixia

iOS – Surge 5

iPhone / iPad
  1. Download Surge 5 from App Store.
  2. Press "OK" → click "Default.conf".
  3. "IMPORT" → Download Profile from URL.
  4. Insert URL: https://dash.gtps.cloud/ios/852
  5. Press "SETUP" & allow VPN.
  6. Open Growtopia & login!

Android – PowerTunnel

No root
  1. Download from GitHub releases.
  2. Open → menu → Hosts settings.
  3. Paste URL:
    https://dash.gtps.cloud/android/852
  4. Tap "Connect" & enjoy.

Windows – Hosts Edit

PC / Laptop
  1. Win+R →

    C:\Windows\System32\drivers\etc

  2. Edit hosts as Admin → remove all & paste:
    5.39.46.39 growtopia1.com
    5.39.46.39 growtopia2.com
  3. Save → launch Growtopia.
